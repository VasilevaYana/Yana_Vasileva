# Решение для соревнования по прогнозированию ценовых интервалов

**Автор:** Vasileva Yana

## Описание подхода

В основе решения лежит ансамбль из 5 нейронных сетей (MLP), обученных с использованием 5-Fold Cross-Validation.

Ключевые особенности решения:
1.  **Целевая метрика:** Вместо стандартных функций потерь (MSE/MAE), была реализована кастомная **Soft-IoU Loss**. Это позволило модели напрямую оптимизировать метрику соревнования (Intersection over Union) в процессе градиентного спуска.
2.  **Архитектура:** Простая, но эффективная полносвязная сеть (MLP) с BatchNorm и Dropout, которая предсказывает не сами границы, а **середину** и **ширину** логарифмированного ценового интервала.
3.  **Пост-обработка:** Для увеличения шанса пересечения с истинным интервалом, предсказанная ширина итогового интервала была увеличена на 5% (`WIDTH_MULTIPLIER = 1.05`).
4.  **Воспроизводимость:** Решение полностью упаковано в Docker-контейнер, что гарантирует идентичную работу на любой машине.

## Как запустить решение

### Требования
- Установленный [Docker](https://www.docker.com/products/docker-desktop/)

### Инструкция по запуску

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/ВАШ_ЛОГИН/vasileva-yana.git
    cd vasileva-yana
    ```

2.  **Поместите данные:**
    Скопируйте файлы `train.csv`, `test.csv` и `sample_submission.csv` в папку `data`.

3.  **Соберите Docker-образ:**
    *Эта команда создаст образ с именем `vasileva-yana-solution`.*
    ```bash
    docker build -t vasileva-yana-solution .
    ```

4.  **Запустите контейнер:**
    *Эта команда запустит обучение и предсказание внутри контейнера. Папки `data` и `results` будут "проброшены" внутрь, чтобы скрипт мог прочитать данные и сохранить результат.*
    ```bash
    docker run --rm -v "$(pwd)/data:/app/data" -v "$(pwd)/results:/app/results" vasileva-yana-solution
    ```

### Ожидаемый результат
После выполнения команды в папке `results` появится файл `submission.csv` с итоговыми предсказаниями.
